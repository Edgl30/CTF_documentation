	|| Formatos: ||
	---------------		
------------------------------------------------------------------------------------
|  |  Ghex:									|  |
|  |  Nos permite visualizar un archivo en formato HEX y BIN			|  |
|  |  Modificarlo, Guardarlo y Buscar en el.     				|  |
|  |				   						|  |
------------------------------------------------------------------------------------
|  |  Hexdump:									|  |
|  |  Buscar info legible en ascii:  						|  |
|  |				hexdump -C archivo.bin | grep -o '[ -~]\{1,\}'  |  |
|  |  Examinar cabecera:  		    hexdump -C imagen.jpg | head -n 20  |  |	
|  |  Buscar patron:  			hexdump -C archivo.bin | grep 'patrón'  |  |
|  |  Convertir HEX a BIN: 							|  |
|  |	      hexdump -v -e '1/1 "%02x"' volcado.hex | xxd -r -p > archivo.bin  |  |
|  |		  								|  |
------------------------------------------------------------------------------------
|  |  Xxd:   									|  |
|  |  Visualizar en hexadecimal:       				xxd archivo.bin |  |
|  |  Convertir Hex a Bin:                 xxd -r -p volcado.hex > archivo.bin  |  |
|  |  Formato de dirección:      		    xxd -r -g 1 -p volcado.hex  |  |
|  |                                                                            |  |
------------------------------------------------------------------------------------
|  |  dd  									|  |
|  |  Restaurar discos:  	    dd if=/ruta/a/imagen.img of=/dev/sdX bs=4M  |  |
|  |  Copiar una parte del archivo: 						|  |
|  |                             dd if=archivo.bin of=parte.bin bs=1 count=512  |  | 
|  |  Recuperar particiones:  							|  |
|  |  			dd if=/dev/sdX of=/ruta/a/particion.img bs=512 count=1  |  |
|  |  ''' Representación HEX y ASCII '''  				   	|  |
